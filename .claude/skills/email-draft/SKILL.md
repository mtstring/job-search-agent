# email-draft

**鉄則（rules/ironclad.md）を絶対に守る**。この skill はその具体手順。

## 手順

1. 元スレッドを `search_threads` で検索（`is:unread` は使わない — rules/ironclad.md）
2. `get_thread` でスレッド全体を取得、To/CC の宛先を確認
3. **全員返信（Reply-All）** で下書き作成
4. 選考進捗に関わる内容なら **skills/status-overview-sync** を発火

## 下書き本文の構成

```
{自分の返信本文}

{送信者名} <{メールアドレス}> {日時}:
> {元メールの本文（純粋なメール本文のみ）}
```

## 絶対に含めない

- **件名・ラベル・注記・ヘッダー**（本文のみが厳守ルール）
- **AI への指示プロンプト**
- **内部メモ・システムテキスト**
- 引用内に**元メール本文以外の何か**

## チェックリスト

- [ ] `is:unread` を使っていない
- [ ] To/CC の全員が宛先に含まれている（Reply-All）
- [ ] 本文のみ（件名・ヘッダー・注記なし）
- [ ] 元メール引用が本文の下にある
- [ ] 引用内に AI 指示・内部メモが混入していない
- [ ] 選考関連なら status-overview-sync を発火した
