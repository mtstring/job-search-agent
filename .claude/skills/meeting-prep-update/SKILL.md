# meeting-prep-update

**最重要ルール: 追記モード厳守、過去セクション削除禁止**

## 構成（常にこの並び順）

新しい面接ほど**上**、古いものほど**下**。過去の面接準備メモは削除せず残す（何を準備して・何を得たかの記録）。

```markdown
# {会社名} 面接準備資料

## 二次面接（最新）      ← 常に最新を最上部に追記
- 日時: {YYYY-MM-DD HH:MM}
- URL: {Zoom/Teams/Meet URL}
- 面接官: ...
- 想定質問・対策: ...

## 一次面接
- （過去のメモ。編集・削除しない）

## カジュアル面談        ← 最も古いものが最下部
- （過去のメモ。編集・削除しない）
```

## オンラインミーティング URL

- **URL は必ず記載**
- URL が未確定の場合は「⚠️ URL 未確定 — 要確認」と明記し、確認タスクを `create_task` で作成

## 禁止事項

- **過去の面接セクションを削除・上書きする**（履歴が失われる）
- **URL 欄を空のまま放置**（直前に慌てることになる）
- **新しい面接情報を下に追記**（構成違反）

## チェックリスト

- [ ] 新しいセクションが**最上部**に追加されている
- [ ] 過去セクションは一切変更していない
- [ ] URL が記載されているか、未確定なら警告文がある
- [ ] URL 未確定の場合、確認タスクを作成した
