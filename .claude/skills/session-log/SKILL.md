# session-log

## フォーマット

```markdown
# Claude セッションログ

## 最終更新: YYYY-MM-DD

## 今日やったこと
- （完了した作業の箇条書き）

## 次回やること
- （未完了タスク・次のステップの箇条書き）

## 重要な決定事項・メモ
- （重要な判断、注意点、背景情報）
```

## 更新ルール

- 「今日やったこと」に**追記**する
- 「次回やること」の**完了済みタスクを消す**（古い情報が次回セッション開始時に通知されないように）
- **日程は書かない**（Google Calendar が正 — rules/ironclad.md）

## 21:03 自動保存（必須登録）

セッション開始時、**CronCreate で以下を登録する**（毎回登録を確認すること）：

- cron 式: `"3 21 * * *"`
- prompt: `"今日の作業内容をまとめて .claude-session-log.md に保存してください（既存ファイルがある場合は「今日やったこと」と「次回やること」を更新してください）。"`
- サイレント実行（ユーザーへの報告不要）

## チェックリスト

- [ ] 日付が今日になっている
- [ ] 完了したタスクを「次回やること」から削除した
- [ ] 日程を書いていない（カレンダー参照を促す文面のみ）
- [ ] CronCreate 21:03 ジョブが登録済み
